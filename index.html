HTML

HTML (Hyper text Markepp Sanguage em inglês ) é uma linguagem de marcação de hipertexto . Ele permite criar e estruturar a aparência de páginas da web.

CSS

CSS ( Cascading Style Shets ) , que significa folha de estilo em cascata, é um mecanismo que adiciona esilos na página do web, como cores, margens, formas, tipos de letras, além de modificar o visual ,adiciona animações, entre outros.

JavaScript

JavaScript é uma linguagem de programação que permite a você criar conteúdo que se atualiza dinamicamente, controlar múltimídias, imagens animadas, e tudo o mais que há de interessante. Ok, não tudo, mas é maravilhoso o que você pode efetuar com algumas linhas de código JavaScript.
